isAdded

() Boolean

Defined in a30_entityState.js:24

Returns:

Boolean:

Whether an entityState instance is EntityState.Added.

Example:

var es = anEntity.entityAspect.entityState;
return es.isAdded();

is the same as

return es === EntityState.Added;

() Boolean
